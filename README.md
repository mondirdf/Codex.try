# Codex.try

موقع بورتفوليو عربي متكامل لمصممة جرافيك (Landing Portfolio) بواجهة حديثة ومتجاوبة.

## المميزات

- صفحة رئيسية احترافية RTL باللغة العربية.
- أقسام واضحة: التعريف، الأعمال، الخدمات، آلية العمل، والتواصل.
- معرض أعمال مع:
  - **فلترة حسب نوع المشروع** (هوية / سوشال / تغليف).
  - **ترتيب عشوائي** سريع للأعمال المعروضة.
- نموذج تواصل مع تحقق (Validation) في الواجهة ورسائل نجاح/خطأ.
- تحسينات وصولية (Accessibility):
  - Skip link.
  - خصائص `aria-*` للقائمة.
  - حالات تركيز واضحة لعناصر التفاعل.
- تأثيرات ظهور ناعمة للعناصر عبر `IntersectionObserver`.

## تشغيل المشروع محليًا

```bash
python3 -m http.server 8000
```

ثم افتح:

`http://localhost:8000`

## بنية الملفات

- `index.html`: البنية والمحتوى.
- `styles.css`: تصميم الواجهة والاستجابة.
- `script.js`: التفاعلات والفلترة والتحقق من النموذج.
